## üîü KEY COMMAND REFERENCES

### Turborepo Commands
```bash
# Initialize new Turborepo
npx create-turbo@latest

# Run all dev servers
pnpm dev

# Build all packages
pnpm build

# Run build with caching
turbo build --filter=<package-name>

# View cache status
turbo build --dry-run
```

### Vite Commands
```bash
# Create new Vite project
npm create vite@latest

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

### CRXJS Commands
```bash
# Create CRXJS project
npm create crxjs@latest

# Dev server
npm run dev

# Build extension
npm run build
```

### pnpm Workspace Commands
```bash
# Install dependencies (entire workspace)
pnpm install

# Add dependency to specific workspace
pnpm add <package> --filter <workspace-name>

# Run script across all workspaces
pnpm -r run <script-name>

# Install from workspace package (local)
pnpm add <local-package> --workspace
```
